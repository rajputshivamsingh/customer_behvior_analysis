Customer Behavior Analysis – Power BI | SQL | Python

This project analyzes customer shopping behavior using Python, SQL, and Power BI. It focuses on identifying customer demographics, spending patterns, category performance, and subscription behavior. The final output is an interactive Customer Behavior Dashboard built in Power BI.

Project Overview

The objective of this project is to perform complete data analysis, including:

Loading and preprocessing the dataset in Python (Pandas)

Performing Exploratory Data Analysis (EDA)

Running analytical queries using SQL

Building a Power BI dashboard to visualize insights

This project demonstrates an end-to-end analytics workflow suitable for real business scenarios.

Dataset Description

The dataset contains:

Customer demographic details

Gender

Age and Age Group

Subscription status (Yes/No)

Product categories (Clothing, Accessories, Footwear, Outerwear)

Purchase amounts

Shipping types

Review ratings

This information helps analyze customer behavior and revenue patterns.

Project Workflow
1. Data Loading in Python

Imported dataset using Pandas

Inspected data types

Checked for missing or incorrect values

2. Data Cleaning

Removed duplicates

Handled missing entries

Standardized column names

Created additional fields (e.g., Age Group, Subscription segments)

3. Exploratory Data Analysis (EDA)

Performed initial analysis to understand:

Total number of customers

Average purchase amount

Customer distribution by gender

Review rating trends

Revenue & sales patterns

4. SQL Analysis

Executed SQL queries to extract insights:

Total customers

Revenue by category

Customer count by subscription

Top performing age group

Category-wise sales
